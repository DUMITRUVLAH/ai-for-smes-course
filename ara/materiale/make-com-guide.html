Ghid Make.com — Email cu CV → Google Sheets

Automatizare completă: când vine un CV pe email, datele ajung singure în Sheets.

⏱️ setup 45–60 min
🧩 5 module
💸 ~$0.01 / CV
📥 ~150 CV-uri/lună gratuit

Ce construim

Când cineva trimite un email cu un CV atașat, Make.com preia automat emailul, extrage din CV numele candidatului, emailul, telefonul și poziția aplicată, și adaugă un rând nou în Google Sheets — fără să atingi nimic manual.

Email primit cu CV.pdf
        ↓
Make.com detectează emailul
        ↓
Extrage datele din CV (cu AI)
        ↓
Adaugă rând în Google Sheets:
[ Ion Popescu | ion@email.com | +373 69 123 456 | Muncitor agricol | Germania ]

Înainte să începi — ce ai nevoie

  • Cont Make.com (gratuit — make.com/register)
  • Cont Google (Gmail + Google Sheets)
  • Cont OpenAI cu credit (minim $5) — pentru extragerea datelor din CV
  • Un email dedicat pentru aplicații (ex: aplicatii@agentia-ta.md) sau eticheta Gmail unde ajung CV-urile

Timp estimat: 45–60 minute prima dată. Cost lunar: gratuit pe Make.com sub ~150 CV-uri/lună; OpenAI ~$0.01 per CV.

0Pregătește Google Sheets

Înainte de Make, creează foaia unde vor ajunge datele.

  • Deschide sheets.google.com → New spreadsheet
  • Redenumește: „Aplicații Candidați 2026"
  • Pe primul rând, scrie capetele de coloană exact așa:
ABCDEF
NumeEmailTelefonPozițieData primiriiFișier CV
De reținut: Make va adăuga câte un rând nou sub rândul 1 pentru fiecare CV. Nu șterge rândul cu capetele de coloană.

1Creează scenariul în Make.com

  • Intră pe make.com → autentifică-te
  • Click Create a new scenario (butonul albastru, stânga sus)
  • Apare o interfață goală cu un cerc în mijloc — primul modul (trigger-ul)
  • Click pe cerc → apare un panou de căutare

2Modulul 1: Gmail — detectează emailul

Ce face: Make verifică Gmail la fiecare 15 minute și pornește scenariul când găsește un email nou cu atașament. Caută Gmail → Watch Emails.

CâmpCe completezi
Connection„Add" → autorizează contul Gmail → dă permisiunile cerute
FolderFolderul/eticheta unde ajung CV-urile (Inbox sau o etichetă „Aplicații")
CriteriaAll email (sau „With attachment")
Max Results1 (un email per rulare — mai sigur la început)
Mark as ReadYes (evită procesarea dublă)
Important: Make monitorizează DOAR folderul ales, nu subfolderele.

3Modulul 2: Gmail — descarcă atașamentul

Ce face: Watch Emails știe că există un atașament, dar nu îl descarcă. Acest modul preia fișierul. Caută Gmail → Get an Email Attachment.

CâmpCe completezi
ConnectionAcelași cont Gmail
Message IDAlege Message ID din modulul 1
Attachment IDAlege Attachments[] > ID din modulul 1
Include file dataYES — obligatoriu, altfel fișierul vine gol
⚠️ Gotcha frecvent: Dacă uiți „Include file data", totul pare să meargă dar AI-ul primește un fișier gol. Verifică mereu că e pe Yes.

4Modulul 3: OpenAI — extrage datele din CV

Ce face: Trimite CV-ul la ChatGPT și primește un JSON cu datele candidatului. Ai nevoie de un API key (platform.openai.com → API Keys). Caută OpenAI → Create a Completion, model gpt-4o-mini. La Messages → Add item → User, pune promptul:

Ești un asistent HR. Extrage din CV-ul de mai jos următoarele câmpuri
și returnează DOAR un JSON valid, fără explicații.

{
  "nume": "",
  "email": "",
  "telefon": "",
  "pozitie_aplicata": "",
  "ani_experienta": ""
}

Dacă un câmp nu există, pune valoarea "negăsit".

CV:
[mapează aici variabila Data din modulul 2 — conținutul fișierului]
PDF scanate: Dacă CV-ul e o imagine, gpt-4o-mini nu o poate citi. Folosește gpt-4o și content type „File" (~$0.03/CV).

5Modulul 4: JSON — parsează răspunsul

Ce face: Transformă textul JSON al AI-ului în variabile separate. Caută JSON → Parse JSON. La „JSON String" mapează Result/Content din modulul OpenAI; la „Data Structure" → Generate, lipește un exemplu:

{
  "nume": "Ion Popescu",
  "email": "ion@email.com",
  "telefon": "+373 69 123 456",
  "pozitie_aplicata": "Muncitor agricol",
  "ani_experienta": "3"
}
⚠️ Gotcha: Câmpurile din Parse JSON nu apar în modulele următoare până nu rulezi scenariul o dată cu date reale. La primul test mergi mai departe oricum, apoi revii și mapezi.

6Modulul 5: Google Sheets — adaugă rândul

Caută Google Sheets → Add a Row, alege fișierul „Aplicații Candidați 2026", apoi mapează coloanele:

ColoanaVariabila mapată
A — Numenume din Parse JSON
B — Emailemail din Parse JSON
C — Telefontelefon din Parse JSON
D — Pozițiepozitie_aplicata din Parse JSON
E — Data primiriiDate din Gmail (Watch Emails)
F — Fișier CVAttachment Name din Gmail

7Testează scenariul

  • Click Run once (triunghiul de play, stânga jos)
  • Trimite-ți un email de test cu un CV atașat
  • Așteaptă ~30 secunde → verifică fiecare modul (balonul cu numărul de deasupra)
  • Dacă e verde → deschide Sheets și verifică rândul nou

Dacă ceva nu merge:

SimptomCauzăSoluție
Modulul 2 dă fișier gol„Include file data" pe NoPune pe Yes
OpenAI dă text, nu JSONPrompt prea slabAdaugă „Returnează DOAR JSON"
Parse JSON fără câmpuriN-ai rulat cu date realeRulează o dată, apoi mapează
Coloane greșite în SheetsMapare incorectăReverifică Pasul 6

8Activează scenariul

  • Click switch-ul OFF/ON (stânga jos) → devine ON
  • Setează intervalul: ceasul de lângă modulul Gmail → 15 minutes (minim pe gratuit)
  • Scenariul rulează acum automat la fiecare 15 minute

Cum extinzi mai departe (opțional)

  • Filtru pe subiect: procesează doar emailuri cu „CV"/„aplicație" — modul Filter între Watch Emails și Get Attachment
  • Notificare Slack/Teams: după rând în Sheets → mesaj „Candidat nou: [Nume] pentru [Poziție]"
  • Google Drive: înainte de OpenAI → „Upload a File" → CV-ul se salvează organizat
  • Email de confirmare: după Sheets → Gmail „Send an Email" către candidat
