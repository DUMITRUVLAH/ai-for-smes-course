AI Agents cu VS Code + Claude

În 2 ore, fiecare participant pleacă cu un agent AI funcțional, construit de el, care lucrează singur pe un task real din munca lui: citește fișiere, folosește unelte, produce livrabile — nu doar răspunsuri în chat. Fără să fii programator.

Ce înveți în 2 ore

de la primul prompt la un agent care lucrează singur
🤖
1
Ce e un agent AI — fără cod
Diferența dintre chat și agent: bucla de tool-use, cum „gândește" și cum decide singur ce unealtă folosește.
💻
2
VS Code + Claude — setup
Instalezi și configurezi mediul de lucru: VS Code, extensia Claude, primul folder de proiect. Totul pe laptopul tău.
📜
3
CLAUDE.md — legea agentului
Fișierul de instrucțiuni care transformă un chat generic într-un coleg care știe regulile, contextul și stilul tău.
🧩
4
Skills & agenți specializați
Abilități reutilizabile în fișiere .md, agenți din marketplace, când folosești un skill vs. un agent întreg.
🔌
5
Conectare la datele tale — MCP
Agentul citește Google Drive, Gmail, fișierele locale — și produce rapoarte din mai multe surse deodată.
🛠️
6
Construiești agentul TĂU
Hands-on pe un task real din munca ta: alegi problema, construiești agentul, îl vezi lucrând cap-coadă.

Agenda celor 2 ore

0:00–0:20
Ce e un agent și de ce contează

Chat vs. agent, bucla de tool-use, exemple reale de agenți care lucrează singuri în companii.

0:20–0:40
Setup: VS Code + Claude

Instalare, configurare, primul folder de proiect și primul dialog cu agentul pe fișierele tale.

0:40–1:10
CLAUDE.md, skills și instrucțiuni

Scrii „legea agentului" pentru task-ul tău: reguli, context, format de livrabil. Adaugi un skill gata făcut.

1:10–1:50
Hands-on: agentul tău, pe task-ul tău

Fiecare participant construiește un agent pe o problemă reală din munca lui — cu suport 1-la-1 de la trainer.

1:50–2:00
Demo-uri + pașii următori

Fiecare arată ce a construit; ce automatizezi săptămâna viitoare și cum scalezi în echipă.

Pentru cine e

✓Angajați și echipe non-tehnice — finanțe, HR, marketing, operațiuni — care vor să automatizeze task-uri repetitive.
✓Manageri care vor să vadă concret ce pot face agenții AI înainte să investească în ei.
✓Oameni care au folosit deja ChatGPT/Claude în browser și vor pasul următor: AI care lucrează, nu doar răspunde.
✓Fără cunoștințe de programare — tot ce construim e ghidat pas cu pas, în limbaj natural.
