🟧 Cele 3 moduri de lucru

Același Claude, trei feluri de a-l folosi — de la conversație simplă la unelte construite. Începi cu Chat și urci pe măsură ce ai nevoie de mai mult.

💬

Conversație

Cel mai bun la documente lungi: contracte, extrase, facturi PDF. Punctul de pornire pentru oricine. Aici rulează Projects, Skills și Gems.
  • Rezumă și verifică un contract sau un set de termeni
  • „Explică-mi clauza asta și ce risc creează"
  • Drafturi de email, proceduri, răspunsuri la furnizori

  • Zeci de facturi → un spreadsheet structurat

  • Integrări între aplicații și surse de date
  • Unelte interne pe care le rulezi de câte ori vrei

💬 Chat clasic vs. Cowork

Diferența fundamentală: cine execută pașii. În chat tu conduci; în Cowork dai obiectivul și Claude îl duce la capăt.

💬 Chat clasic
  • Tu conduci fiecare pas: ceri, citești, ceri din nou
  • Prompt → răspuns → tu execuți → prompt următor
  • Ideal pentru drafturi, întrebări, analize rapide
⚡ Cowork
  • Tu dai obiectivul — Claude planifică, execută, livrează
  • Mai mulți pași automat, fără intervenție pas cu pas
  • Ideal pentru task-uri repetitive, extragere date în masă

Ce poate face Cowork pentru tine

Task-uri pe care le faci azi manual — Cowork le duce singur la capăt.

🧾
Extragere date în masă
Zeci de facturi sau bonuri → un spreadsheet structurat (furnizor, sumă, TVA, dată), fără introducere manuală.
📁
Organizare dosare
Scanează un folder dezorganizat → propune o schemă → redenumește și sortează după aprobarea ta.
📧
Gmail + Excel — concedii
Preia cererea de concediu din Gmail → o înregistrează automat în tabelul de concedii din Excel.
📊
Rapoarte combinate
Date din Gmail + Excel + documente → un raport consolidat, în format standard.
💼 Cowork e disponibil în Claude for Work (~$25/utilizator/lună) — DPA inclus, datele nu sunt folosite pentru training. Important pentru date financiare.

📄 Skills — generează documente reale, nu text

Instrucțiuni + template → fișier real (Word / PDF / Excel), în formatul firmei tale, de fiecare dată. Nu primești text de copiat, ci un document gata de folosit.

📝
Act de primire-predare
Date tranzacție + persoane → draft de act .docx gata de semnat, cu structura ta.
🔎
Document din mai multe surse
Pe baza mai multor documente → extrage informația și generează un document consolidat.
📊
Skill „Analiză financiară"
Skill gata pregătit (ex. financial-analyst) → analiză standardizată din situațiile financiare.

Cum generezi un document real — pas cu pas

  1. Intri într-un Project în Claude
  2. Creezi sau alegi un Skill (instrucțiuni + template)
  3. Completezi inputurile cu date reale
  4. Generezi documentul → descarci .docx
  5. Verifici structura → trimiți
🧩 Skill-uri publice gata făcute găsești pe GitHub (ex. financial-analyst). Rezultatul: un document care arată ca al firmei, generat în minute.

📁 Projects — un dosar permanent cu context

Încarci o dată documentele și instrucțiunile, iar Claude le „ține minte" în toate conversațiile din acel Project. Nu mai reîncarci contextul de fiecare dată.

🗂️
Context o dată, folosit mereu
Politici interne, plan de conturi, modele de contract, situații financiare — încărcate o singură dată în Project.
👤
Un dosar per client / lună
Un Project pentru fiecare client, dosar sau perioadă. Toate conversațiile pornesc din același context corect.
🔗
Baza pentru Skills
Projects + Skills = output consistent cu standardul firmei, fără să reexplici de fiecare dată ce vrei.

📊 Claude în Excel — analiză financiară fără formule

Claude citește o foaie de calcul și răspunde în limbaj normal, generează altele noi — fără VLOOKUP, fără tabele pivot scrise manual.

💬
Întrebi foaia de calcul
„Care e soldul net pe luna trecută?", „Cine sunt top 5 furnizori după valoare?" — răspuns direct, fără formule.
📈
Construiește cashflow & indicatori
Dintr-un extras sau o listă de plăți → cashflow structurat, burn rate, runway, scenarii.
🧹
Curăță & restructurează
Tabele dezordonate → date curate, coloane standard, gata de raport. Generează și fișiere noi.

🔌 Conectori (MCP) — Claude acționează, nu doar răspunde

Cu MCP, Claude nu doar citește — acționează: trimite emailuri, adaugă fișiere, creează taskuri. Conectat la aplicațiile pe care le folosești deja.

❌ Fără MCP
  • AI izolat → copiezi/lipești manual din Drive, email, calendar
  • Timp pierdut pe transfer manual de context
✅ Cu MCP
  • „Ia contractul din Drive și actualizează valorile" → o face
  • „Trimite emailul de confirmare clientului din Gmail" → îl trimite
  • „Adaugă în Calendar ședința de joi la 10:00" → adaugă
🔗 Conectori: Gmail · Google Drive · Calendar · Trello · Slack · Excel / Sheets + sute de alți conectori. Tot aici intră extensia Claude pentru Chrome — Claude vede pagina pe care ești și acționează în context.
